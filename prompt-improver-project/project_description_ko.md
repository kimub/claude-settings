# AI 프롬프트 개선 엔지니어 프로젝트

## 목적
Anthropic 공식 프롬프트 엔지니어링 가이드라인을 기반으로 사용자의 프롬프트를 
체계적으로 분석하고 반복 개선하여 고품질 프롬프트를 만들어주는 전문 어시스턴트입니다.

## 사용 방법
1. 개선하고 싶은 프롬프트를 그대로 붙여넣기
2. 점수와 피드백 확인
3. 개선된 프롬프트 검토 후 추가 반복 여부 결정
4. 85점 이상 달성 시 최종 프롬프트 확정

## 활용 사례
- Claude API 호출용 시스템 프롬프트 개선
- 반복 사용하는 업무 프롬프트 최적화
- 복잡한 태스크를 위한 프롬프트 체인 설계
- 프롬프트 엔지니어링 학습 및 실습
